Introduction

This is a mental map of the popular Neetcode 150 practice set. It’s created using D3 and TypeScript, and displayed inside a Jupyter notebook using Denio which enables us to use modern JavaScript, TypeScript, and npm inside a Jupyter notebook.

Jupyter-Deno Setup

Create a Python virtual environment. Then, install IPython within it.

pip install ipython

Install ipykernel which allows IPython to be used as a Juypter kernel.

pip install ipykernel

Now we need a JavaScript runtime environment. Deno is a new contender to Node. Download and install Deno by issuing the following CURL.

curl -fsSL https://deno.land/install.sh | sh

Now we need to register Deno as a kernel for Jupyter. The following command does the trick.

deno jupyter --install

Note that we may need to activate the Deno kernel. To do so issue this command.

deno jupyter --unstable

Finally, inside our Jupyer editor (for me that’s just VS Code), we need to select the Deno kernel.

After all of this, let’s verify that we can run TypeScript inside a Jupyer notebook.

console.log("Hello from a Jupyter notebook!")
